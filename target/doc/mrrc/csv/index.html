CSV serialization of MARC records.

This module provides conversion of MARC records to CSV (Comma-Separated Values) format, suitable for import into spreadsheet applications and data analysis tools.

§Examples

ⓘ
use mrrc::{Record, Field, Leader, csv};

let mut record = Record::new(Leader::default());
let mut field = Field::new("245".to_string(), '1', '0');
field.add_subfield('a', "Title".to_string());
record.add_field(field);

let csv = csv::records_to_csv(&[record])?;
println!("{}", csv);

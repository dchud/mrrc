Record boundary detection using 0x1D delimiters for parallel processing.

This module provides optimized scanning of MARC record boundaries using the SIMD-accelerated memchr crate to locate 0x1D (record terminator) bytes. Boundaries are returned as (offset, length) tuples for use in parallel parsing pipelines.

§Example

use mrrc::boundary_scanner::RecordBoundaryScanner;

let buffer = b"...binary MARC data...";
let mut scanner = RecordBoundaryScanner::new();
let boundaries = scanner.scan(buffer)?;

for (offset, len) in boundaries {
    println!("Record at offset {} with length {}", offset, len);
}

RECORD_TERMINATOR 🔒
The byte value that terminates MARC records (ISO 2709). In ISO 2709 format, records end with 0x1D (not 0x1E, which is the field terminator).

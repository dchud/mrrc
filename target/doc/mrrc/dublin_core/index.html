Dublin Core serialization of MARC records.

This module provides conversion of MARC records to Dublin Core metadata, a simple and widely-adopted 15-element metadata schema used for resource discovery.

The Dublin Core Metadata Element Set (DCMES) consists of:

  • Title (dc:title)
  • Creator (dc:creator)
  • Subject (dc:subject)
  • Description (dc:description)
  • Publisher (dc:publisher)
  • Contributor (dc:contributor)
  • Date (dc:date)
  • Type (dc:type)
  • Format (dc:format)
  • Identifier (dc:identifier)
  • Source (dc:source)
  • Language (dc:language)
  • Relation (dc:relation)
  • Coverage (dc:coverage)
  • Rights (dc:rights)

§Examples

ⓘ
use mrrc::{Record, Field, Leader, dublin_core};

let mut record = Record::new(Leader::default());
let mut field = Field::new("245".to_string(), '1', '0');
field.add_subfield('a', "Title".to_string());
record.add_field(field);

let dc_xml = dublin_core::record_to_dublin_core(&record)?;
println!("{}", dc_xml);

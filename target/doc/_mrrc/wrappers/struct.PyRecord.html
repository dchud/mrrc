Expand description

Python wrapper for a Record

A MARC bibliographic record is the fundamental unit of MARC data. It consists of a leader (24-byte header), control fields (000-009), and data fields (010 and above).

§Examples

import mrrc
leader = mrrc.Leader()
leader.record_type = 'a'
leader.bibliographic_level = 'm'

record = mrrc.Record(leader)
record.add_control_field('001', '12345')

field = mrrc.Field('245', '1', '0')
field.add_subfield('a', 'Example Title')
record.add_field(field)

print(f"Title: {record.title()}")

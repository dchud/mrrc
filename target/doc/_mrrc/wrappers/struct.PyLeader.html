Expand description

Python wrapper for a MARC Leader (24-byte record header)

The MARC leader is a 24-byte fixed-length field at the start of every MARC record. It contains metadata describing the record’s structure, content type, and encoding.

§Examples

import mrrc
leader = mrrc.Leader()
leader.record_type = 'a'  # Language material
leader.bibliographic_level = 'm'  # Monograph

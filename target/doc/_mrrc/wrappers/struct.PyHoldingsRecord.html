pub struct PyHoldingsRecord {

Expand description

Python wrapper for a MARC Holdings Record (Type x/y/v/u)

Holdings records are used to maintain inventory and location information. They use the same ISO 2709 binary format as bibliographic records but organize fields by functional role (locations, enumeration, notes, etc.).
